Chat with a Designated Pin
Some Notes
Slightly different from the previous Start Chat code, this allows you, the developer, to hardcode in a pin to chat with.
If the pin specified doesn't exist, the user of the app will be prompted to add them.

The Code:
function startChatWith() {
blackberry.invoke.invoke({
action: 'bb.action.BBMCHAT',
uri: 'pin:32693EF6'
});
}

Do it!
onclick="startChatWith()"
